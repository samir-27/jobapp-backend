A backend service for a job portal application built with NestJS and MongoDB, handling users, companies, jobs, resumes, and admin operations with secure role-based access.

# Features

- User authentication & profile management

- Company registration & company profile management

- Job CRUD (create, update, delete, list jobs)

- Resume upload & job application system

- Admin panel APIs (manage users, companies, jobs)

- Role-based access control (User / Company / Admin)

- Cloudinary integration for resume/file uploads

- Scalable and modular NestJS architecture






## Tech Stack

- NestJS

- MongoDB

- Mongoose

- JWT Authentication

- Cloudinary

- TypeScript

## Run Locally

Clone the project

```bash
  git clone https://github.com/samir-27/jobapp-backend.git
```


Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
